# XinovaSU Landing Page

白色、响应式的 XinovaSU 独立静态落地页，含桌面与移动端布局、可访问的移动菜单、功能卡交互以及减少动态效果偏好支持。

## 本地预览

```bash
npm ci
npm run dev
```

验证：

```bash
npm test
npm run build
npm run test:e2e
```

## 公网发布

推送到 `main` 后，GitHub Actions 会构建并部署到 GitHub Pages。仓库的 Pages 来源应设为 **GitHub Actions**；站点地址为：

`https://kusesad-1122.github.io/xinovasu-landing/`
